# Contributing to FMP Monograph

This repository is the monograph branch of the Fractal Metascience Paradigm.

Use it for:

- monograph structure repair;
- bibliography and citation alignment;
- chapter cleanup and language normalization;
- detox of duplicate or broken publication artifacts.

## Before contributing

Read:

- `README.md`
- `CITATION.cff`
- `SECURITY.md`
- `CODE_OF_CONDUCT.md`

When repository rules or publication governance are in question, use
`terra-legal` as the donor layer.

## What belongs here

Good contributions include:

- chapter and section cleanup;
- bibliography synchronization;
- monograph package repair;
- improvement of readable public entry points.

## What does not belong here

Avoid:

- using the repository as a generic Terra dump;
- creating duplicate chapter variants without clear reason;
- leaving damaged encodings or weak metadata in place.

## Working sequence

1. Read the target section and nearby monograph files.
2. Check whether another stronger version already exists.
3. Repair encoding, citation, and structure before expansion.
4. Preserve the repository as a monograph branch, not a mixed archive.

## Security

If you encounter a release, token, or publication-integrity problem, report it
through `SECURITY.md`.
